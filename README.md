# Formula 1 companion

A MERN based web application which aims to provide information about races during a particular season, constructor and driver details as well as standings. Currently, only race and circuit details are included in this skeleton platform. These details were collected from the jolpi.ca F1 API. Other modules such as standings and results of previous races will be added. The eventual goal is to have a fully polished and functional one stop platform for essential race related information throughout a season.

## Tech Stack

- Node
- React
- Tailwind CSS
- Express
- Axios
- MongoDB
